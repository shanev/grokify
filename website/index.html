The extension will be available on the Chrome Web Store soon. For now, you can install it manually:

üõ†Ô∏è Manual Installation

  1. Download or clone the repository from GitHub
  2. Navigate to the extension folder
  3. Run bun install to install dependencies
  4. Run bun run build to build the extension
  5. Open Chrome and go to chrome://extensions/
  6. Enable "Developer mode" (toggle in top right)
  7. Click "Load unpacked"
  8. Select the build/chrome-mv3-prod folder
  9. Start browsing - all Wikipedia links will now point to Grokipedia!
